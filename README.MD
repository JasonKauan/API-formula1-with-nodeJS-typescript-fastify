Ideal para quem quer uma base leve e moderna para APIs REST.

## 🚀 Tecnologias usadas

- [Node.js](https://nodejs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Fastify](https://www.fastify.io/)
- [npm](https://www.npmjs.com/) (gerenciador de pacotes)

---

## 📦 Como rodar o projeto

Clone o repositório:
```bash
git clone https://github.com/JasonKauan/API-formula1-with-nodeJS-typescript-fastify.git
```

Entre na pasta do projeto:
```bash
cd API-formula1-with-nodeJS-typescript-fastify
```

Instale as dependências:
```bash
npm install
```

Rode o servidor de desenvolvimento:
```bash
npm run dev
```
